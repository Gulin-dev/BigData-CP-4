# BigData-CP-4
Техническое задание:

Задание 1
Создайте таблицу ST.XXXX_SALARY_HIST, где XXXX - ваш идентификатор. 
В таблице должна быть SCD2 версия таблицы DE.HISTGROUP с полями:  
PERSON
CLASS
SALARY
EFFECTIVE_FROM
EFFECTIVE_TO
Для формирования технических полей исторической таблицы используйте оконные функции. Поскольку в датах нет момента времени, берите интервал - 1 день

Задание 2
Возьмите в работу таблицы ST.XXXX_SALARY_HIST и DE.SALARY_PAYMENTS. Напишите SQL скрипт, выводящий таблицу платежей сотрудникам. В таблице должны быть поля PAYMENT_DT, PERSON, PAYMENT, MONTH_PAID, MONTH_REST. 
Результат выполнения сохраните в таблицу ST.XXXX_SALARY_LOG.

MONTH_PAID - суммарно выплачено в месяце,
MONTH_REST - осталось выплатить за месяц.

Проверяется в первую очередь понимание как соединять фактовую таблицу с SCD2 таблицей (нельзя все расчеты сделать над DE.SALARY_PAYMENTS, ведь работнику могут недоплатить или переплатить).

В ответе приложите SQL скрипт, таблица ST.XXXX_SALARY_LOG должна быть заполнена.
